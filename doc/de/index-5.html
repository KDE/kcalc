5. Tastenkürzel

F1

ruft die Hilfe auf.

F2

öffnet den Konfigurationsdialog.

F3

schaltet in den Trigonometrie-Modus um.

F4

schaltet in den Statistik-Modus um.

Pfeil oben

zeigt das vorherige Element des Result Stacks an.

Pfeil unten

zeigt das nächste Element des Result Stacks an.

Bild oben

löscht.

Bild unten

löscht alles.

ALT + d

setzt die Winkeleinheit auf Grad.

ALT + r

setzt die Winkeleinheit auf Radiant.

ALT + g

setzt die Winkeleinheit auf Neugrad.

ALT + h

setzt das Zahlensystem auf Hexadezimal.

ALT + e

setzt das Zahlensystem auf Dezimal.

ALT + o

setzt das Zahlensystem auf Oktal.

ALT + b

setzt das Zahlensystem auf Binär.

s

Sin

c

Cos

t

Tan

n

Ln berechnet den Logarithmus zur Basis e.

l

Log berechnet den Logarithmus zur Basis 10.

i

Inv, Invers; tippen sie z.B. ``i'' ``s'' um arcsin (x) zu berechnen.

\\

+/-

[

x^2 *************************************

^

x^y

=

=

!

x! berechnet die Faktorielle.

<

Lsh schiebt nach links.

>

Inv Lsh schiebt nach rechts.

&

And berechnet die Konjunktion (logisches UND).

*

X berechnet das Produkt.

\

/ berechnet den Quotienten.

d

Dat gibt (im Statistik-Modus) einen Wert ein.

o

Or berechnet die Disjunktion (logisches ODER).

i o

Inv Or berechnet die Antivalenz (ausschließendes ODER, Exklusiv-Oder)

r

1/x berechnet den Reziproken Wert.

Enter

=

Return

=

Backspace

AC löscht alles.

Del

AC löscht alles.

Esc

C löscht.

h

Hyp schaltet in den ``Hyperbolischen'' Modus. Z.B. berechnet Hyp Sin den Sinus Hyperbolicus: sinh(x).
